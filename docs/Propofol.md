# Propofol  
= Disoprivan®
(Stand: 24.03.2026)

---
## 🚨 CAVE / MERKE 💡
- 🤰**Schwangerschaft:** nur bei zwingender Indikation; keine generelle KI  
- 🍼 **Stillzeit:** keine Stillpause nötig (geringe Milchkonzentration)  
- bei Polytrauma oder Ähnlichem
	→ ggf. [Ketamin](Ketamin.md) oder [Etomidate](Etomidate.md) bevorzugen 
		→ weniger kreislaufdepressiv  
- Propofol-Infusionssyndrom ([PRIS](PRIS.md)) bei Langzeitanwendung möglich  
- Vorsicht bei Leber-/Niereninsuffizienz:  
	- Leber: verminderte Metabolisierung  
		→ verlängerte Wirkung  
	- Niere: reduzierte Elimination der Metabolite  
		→ Kumulationsrisiko  
- strikte Aseptik nötig (Kontaminationsrisiko der Emulsion!)

---
## 💊 DOSIERUNG

#### Narkoseeinleitung
| Patientengruppe     | Dosis (mg/kg) |
| ------------------- | ------------- |
| 👨🏻👩🏼 Erwachsene | 1,5–2,5       |
| 👶 Baby             | 2,5–3,5       |
| 👴🏻👵🏼 Geriatrie  | 0,5–1,5       |

#### Narkoseaufrechterhaltung
| Anwendung           | Dosis (µg/kg/min) | Entspricht (mg/kg/h) |
| ------------------- | ----------------- | -------------------- |
| 👨🏻👩🏼 Erwachsene | 75–150            | 4,5–9                |
| 👶 Baby             | 125–300           | 7,5–18               |

#### TCI (Target Controlled Infusion)
- Modelle: **Marsh** (Erwachsene), **Schnider** (Berücksichtigung Alter/Körperbau)  
- Zielwerte: 2–6 µg/ml    
- Vorteil: präzisere Steuerung, geringere Überdosierung

## ⚗️ PHARMAKOLOGIE

| Parameter             | Wert                                                  |
| --------------------- | ----------------------------------------------------- |
| WE ⏱️                 | 15–45 s                                               |
| WD ⏳                  | 5–10 min                                              |
| Kontext-sensitive HWZ | bis zu mehrere Stunden bei Langzeitsedierung          |
| Metabolisierung       | v. a. Leber, zusätzlich extrahepatisch (Niere, Lunge) |
| Elimination           | renal als Metabolite                                  |
| Energiegehalt         | ca. **1,1 kcal/ml** (Fettlösung)                      |

## ✨ WIRKUNG
- verstärkt **[GABA](GABA.md)-A** 
	→ [GABA](GABA.md) = inhibitorischer Neurotransmitter  
- hemmt indirekt **[NMDA](NMDA.md)** 
	→ [NMDA](NMDA.md) = exzitatorischer Neurotransmitter  
- moduliert Na⁺- und K⁺-Kanäle 
	→ führt durch Hyperpolarisation der Nervenzellen zu verringerter Erregbarkeit  
- Amnesie, [Anxiolyse](Anxiolyse.md), Bewusstlosigkeit  
- Senkung von zerebralem Blutfluss (CBF) und zerebralem Metabolismus (CMRO₂) → intrakranieller Druck ↓  
- tierexperimentell neuroprotektiv → klinische Relevanz fraglich, bisher keine eindeutigen Human-Daten  

---
## ✅ PRO
- angenehmes Einschlafen & Aufwachen  
- antiemetisch → weniger [PONV](PONV.md)  
- Bronchodilatation  
- gut für LAMAs → Unterdrückung oropharyngealer Reflexe  
- Langzeitanwendung möglich → aber CAVE: [PRIS](PRIS.md)  

---
## 🌀 NEBENWIRKUNGEN (CONTRA)
- Awareness 
	→ Neuromonitoring empfohlen
	→ ggf. low-dose Sevofluran  
- Injektionsschmerz (~20 %)  
- kardio- & atemdepressiv    
- Fettbelastung (Triglyzeride)  
- [PRIS](PRIS.md):  
	- Frühsymptome: metabol. Azidose, HRST, Herzversagen  
	- Spätsymptome: Rhabdomyolyse, Nierenversagen  
	- Risikofaktoren: >4 mg/kg/h, >48 h, kritisch kranke Kinder  
	- Letalität bis 50 %‼️  

---
## ❌ KONTRAINDIKATIONEN
- Schock  
- Allergie auf:  
	- Propofol  
	- Ei, Soja, Erdnuss (theoretisch, klinisch meist irrelevant)  
- Kinder < 1 Monat  
- Langzeitsedierung bei < 16 Jährigen   
- Schwangerschaft (relativ, nur bei zwingender Indikation)

## 📊 VERGLEICH
| Substanz                       | Kreislaufwirkung | Atemdepression | Besonderheiten                             |
| ------------------------------ | ---------------- | -------------- | ------------------------------------------ |
| [Etomidate](Etomidate.md)      | neutral          | gering         | Cortisol-Suppression, Myoklonien           |
| [Propofol](Propofol.md)        | RR↓, HR↓         | ausgeprägt     | antiemetisch, Lipidformulierung            |
| [Thiopental](Thiopental.md) | RR↓              | mittel         | Histaminfreisetzung, lange HWZ             |
| [Midazolam](Midazolam.md)      | leicht RR↓       | mittel         | Amnesie, anxiolytisch                      |
| [Ketamin](Ketamin.md)          | RR↑, HR↑         | gering         | Analgesie, Dissoziation, Bronchodilatation |

---
## 🧪 CIPROFOL
- neu entwickeltes Hypnotikum (China)  
- strukturell verwandt mit Propofol, höhere Potenz  
- Ziel: geringere Dosis, weniger kardiodepressiv  
- noch keine breite Zulassung in Europa/USA  

| Parameter         | Propofol                   | Ciprofol                  |
| ----------------- | -------------------------- | ------------------------- |
| Potenz            | Standard                   | ca. 4–5× stärker          |
| Dosis (Induktion) | 1,5–2,5 mg/kg              | 0,4–0,5 mg/kg             |
| Kreislaufwirkung  | deutlich ↓ MAP             | weniger kardiodepressiv   |
| Metabolismus      | hepatisch + extrahepatisch | ähnlich, aber effizienter |
| Zulassung         | 🌍 weltweit etabliert      | bisher v. a. China 🇨🇳   |
| Besonderheiten    | PRIS-Risiko bei Langzeit   | geringere Lipidlast       |
