# GCS
= Glasgow Coma Scale

> [!warning] 🚨 KURZ & WICHTIG 💡
> - quantitative Beurteilung der Bewusstseinslage  
> - gültig für Erwachsene & Kinder **≥ 5 Jahre**  
> - immer **E – V – M getrennt** dokumentieren  
> - **Motorik** ist der **stärkste** Prädiktor  
> - **GCS ≤ 8** → Risiko für Verlust der Schutzreflexe → ITN  

---

## 🧩 EINTEILUNG

### 👁️ Augenöffnung (E - Eyes)

| Score | Erwachsene | Kinder |
|-------|------------|--------|
| **4** | spontan | spontan |
| **3** | auf Ansprache | auf Schreien |
| **2** | auf Schmerzreiz | auf Schmerzreiz |
| **1** | keine | keine |

### 🗣️ Verbale Reaktion (V - verbal)

| Score | Erwachsene                   | Kinder     |
| ----- | ---------------------------- | ---------- |
| **5** | orientiert                   | brabbeln   |
| **4** | desorientiert                | tröstbar   |
| **3** | inadäquate Worte / Wortsalat | untröstbar |
| **2** | Laute                        | Laute      |
| **1** | keine                        | keine      |

### ✋ Motorische Reaktion (M - Motorik)

| Score | Erwachsene | Kinder |
|-------|------------|--------|
| **6** | befolgt Aufforderungen | befolgt Aufforderungen |
| **5** | gezielte Schmerzabwehr | gezielte Schmerzabwehr |
| **4** | ungezielte Schmerzabwehr | ungezielte Schmerzabwehr |
| **3** | Beugesynergismen | Beugesynergismen |
| **2** | Strecksynergismen | Strecksynergismen |
| **1** | keine | keine |

---

## 📊 INTERPRETATION

- **GCS 13–15** → leicht  
- **GCS 9–12** → moderat  
- **GCS ≤ 8** → schwer, potenziell **intubationspflichtig**  
- **Abfall ≥ 2 Punkte** → akute neurologische Verschlechterung  
- **Asymmetrische Motorik** → strukturelle Läsion bis Beweis des Gegenteils  

**Formel:**  
**GCS = E + V + M** (min. 3, max. 15)

**Dokumentation:**  
- immer **E–V–M einzeln**  
- Reizart angeben (z. B. *Schmerzreiz: supraorbital*)  
- Intubiert: **V = t**  
- Sediert: **nb** statt 1  

**Beispiel:**  
**GCS: E3 Vt M5 (Schmerzreiz supraorbital)**
