# COPD
---
## 💬 KURZ
- Chronisch progrediente, nicht vollständig reversible Obstruktion der kleinen Atemwege durch inhalative Noxen (v. a. Rauchen).  
- Leitsymptome: **AHA‑Trias  
	→ Auswurf  
	→ Husten  
	→ [Atemnot](Dyspnoe.md)**   

---
## 🚨 CAVE / MERKE 💡
- COPD = weltweit vierthäufigste Todesursache  
- ICS nur bei Eosinophilie oder häufigen Exazerbationen  

---
## ❓ URSACHEN
| Ursache                          | Details                                   |
| -------------------------------- | ----------------------------------------- |
| ‼️ Rauchen 🚬                    | ≈ 90 %                                    |
| Luftverschmutzung / Feinstaub 💨 | beruflich & urban                         |
| Infektionen 🦠                   | *H. influenzae, Pneumokokken, Rhinoviren* |
| **α1‑Antitrypsin‑Mangel**        | v. a. bei jungen Patienten                |

---
## ⚙️ PATHOPHYSIOLOGIE
**Inhalative Noxen (v. a. Rauchen)**  
	⬇️  
**chronische Entzündung der kleinen Atemwege**  
	⬇️  
**Zerstörung des Flimmerepithels**  
	⬇️  
**Hypersekretion + gestörte mukoziliäre Clearance**  
	⬇️  
**Husten + forcierte Exspiration**  
	⬇️  
**Bronchialkollaps (v. a. exspiratorisch)**  
	⬇️  
**Obstruktion (FEV1↓, Resistance↑)**  
	⬇️  
**Lungenüberblähung (RV↑)** mit **Emphysem**  
	⬇️  
**Ventilations‑Perfusions‑Mismatch** → **Hypoxie / Hyperkapnie**

---
## 🚑 KLINIK
| Befund | Hinweise |
|--------|----------|
| **AHA‑Trias** | Auswurf – Husten – Atemnot |
| Auskultation | exspiratorisches Giemen/Brummen |
| Hypoxämie | Zyanose, Trommelschlägelfinger |
| Cor pulmonale | Halsvenenstauung, Ödeme, Stauungsleber |

---
## 🧩 EINTEILUNG

### ABE‑Klassifikation
| Gruppe | Symptome (mMRC / CAT)                  | Exazerbationen/Jahr                | Bedeutung               |
| ------ | -------------------------------------- | ---------------------------------- | ----------------------- |
| **A**  | wenig Symptome (mMRC 0–1 oder CAT <10) | 0–1                                | milde Symptomatik       |
| **B**  | viele Symptome (mMRC ≥2 oder CAT ≥10)  | 0–1                                | ausgeprägte Symptomatik |
| **E**  | egal                                   | **≥2** oder **≥1 Hospitalisation** | hohe Exazerbationslast  |

### mMRC‑Dyspnoe‑Skala
| Punkte | Beschreibung                                       |
| ------ | -------------------------------------------------- |
| 0      | [Atemnot](Dyspnoe.md) nur bei starker Belastung    |
| 1      | [Atemnot](Dyspnoe.md) beim schnellen Gehen/bergauf |
| 2      | Geht langsamer als Gleichaltrige, Pausen nötig     |
| 3      | Pause nach 100 m oder wenigen Minuten              |
| 4      | Haus kaum verlassbar, kaum selbstständig           |

**Interpretation:**  
**mMRC ≥2 = klinisch relevante Dyspnoe** 
	→ entspricht Gruppe **B** oder **E**

## 🩺 DIAGNOSTIK

#### 1. Untersuchung
| Befund                   | Bedeutung   |
| ------------------------ | ----------- |
| hypersonorer Klopfschall | Überblähung |
| Giemen/Brummen           | Obstruktion |
| Fassthorax               | Emphysem    |

#### 2. Labor 🧪 

| Parameter         | Interpretation                                   |
| ----------------- | ------------------------------------------------ |
| Entzündungswerte↑ | Infekt                                           |
| BGA               | pO₂↓ = hypoxisch<br>pO₂↓ + pCO₂↑ = hyperkapnisch |
| α1‑Antitrypsin    | bei jungen Patienten                             |

#### 3. Lungenfunktion 🫁 
→ Messung nach Bronchodilatation, nicht in akuter Exazerbation 💡  

| Parameter | Diagnose |
|-----------|----------|
| **Tiffeneau <70 %** | Obstruktion |
| FEV1 >80 % | Grad 1 |
| FEV1 <80 % | Grad 2 |
| FEV1 <50 % | Grad 3 |
| FEV1 <30 %* | Grad 4 |

\* **FEV1 <30 %** *oder* **<50 % + Rechtsherzinsuffizienz** *oder* **<50 % + respiratorische Insuffizienz**

#### 4. Bildgebung 🩻 

| Methode | Befund |
|---------|--------|
| Röntgen | Überblähung, Emphysem, Cor pulmonale |

## 🔀 DIFFERENTIALDIAGNOSEN

#### 1. Lunge 🫁 
| DD                                          | Typische Merkmale                                      | Abgrenzung zu COPD                  |
| ------------------------------------------- | ------------------------------------------------------ | ----------------------------------- |
| [Asthma-Bronchiale](Asthma-Bronchiale.md) | variabel, anfallsartig, oft allergisch                 | reversibel, häufig junges Alter     |
| Pneumonie                                   | Fieber, Husten, fokale RGs                             | akuter Beginn, Infiltrat im Röntgen |
| Pneumothorax                                | plötzliche [Dyspnoe](Dyspnoe.md), einseitig hypersonor | fehlendes Atemgeräusch, Röntgen     |
| Lungenembolie                               | akute [Dyspnoe](Dyspnoe.md), Tachykardie               | D‑Dimer, CT‑Angio                   |
| Bronchiektasen                              | viel Sputum, rezidivierende Infekte                    | CT‑Nachweis                         |
| ILD                                         | Belastungsdyspnoe, trockener Husten                    | restriktive LuFu, HR‑CT             |

#### 2. Herz ❤️ 
| DD               | Typische Merkmale                | Abgrenzung zu COPD     |
| ---------------- | -------------------------------- | ---------------------- |
| Herzinsuffizienz | Beinödeme, Orthopnoe             | BNP↑, Echo             |
| Asthma cardiale  | nächtliche [Dyspnoe](Dyspnoe.md) | kardiale Ursache, Echo |
| ACS              | Thoraxschmerz, Risikofaktoren    | EKG, Troponin          |

## 🏥 THERAPIE

### I. A l l g e m e i n 💬
| Maßnahme             | Nutzen                                              |
| -------------------- | --------------------------------------------------- |
| **Rauchstopp** 🚭    | stärkster prognostischer Effekt                     |
| Patientenschulung 📖 | Atemtechniken, Lippenbremse                         |
| Sport / Reha 💪      | Mortalität↓                                         |
| Impfungen 💉         | **v. a. ab 60 Jahren**: *Influenza*, *Pneumokokken* |

---

### II. M e d i k a m e n t ö s 💊 

#### 1. Grundprinzipien
| Merke | Bedeutung |
|-------|-----------|
| **LAMA > LABA** | bessere Exazerbationsprophylaxe |
| Fixkombis > Einzelsubstanzen | Adhärenz↑ |
| ICS nur bei Eosinophilie >300/µl oder häufigen Exazerbationen | Exazerbationen ↓, Pneumonierisiko ↑ |
| ICS bei GOLD A/B absetzen | Prognose unverändert |

#### 2. ABE‑Therapieschema

| Gruppe | Empfehlung                                                                                                                       |
| ------ | -------------------------------------------------------------------------------------------------------------------------------- |
| **A**  | **Nichts** (wenn es geht)<br>*oder*<br>**SABA** oder **SAMA** bei Bedarf<br>*oder*<br>**LAMA** oder **LABA** Dauermedikation     |
| **B**  | **LAMA + LABA** Dauermedikation                                                                                                  |
| **E**  | **LAMA + LABA** bevorzugt<br>*oder*<br>**LABA + ICS** bei Eos ≥300/µl<br>*oder*<br>**LAMA+LABA+ICS** bei hoher Exazerbationslast |

#### 3. Wirkstoffe 
| Gruppe            | Beispiele                                                     |
| ----------------- | ------------------------------------------------------------- |
| **SAMA**          | *Ipratropium*                                                 |
| **SABA**          | *Salbutamol, Fenoterol, Reproterol, Terbutalin*               |
| **LAMA**          | *Tiotropium, Glycopyrronium, Umeclidinium, Aclidinium*        |
| **LABA**          | *Formoterol, Salmeterol, Indacaterol, Vilanterol, Olodaterol* |
| **ICS**           | *Budesonid, Fluticason, Beclometason, Mometason, Ciclesonid*  |
| **PDE‑4‑Hemmer**  | *Roflumilast*                                                 |
| ***Theophyllin*** | Reserve, geringe therapeutische Breite                        |
| **Mukolytika**    | *Acetylcystein*                                               |
| **Antibiotika**   | *Amoxicillin/Clavulanat, Doxycyclin* (bei Exazerbation)       |

---

### III. N i c h t m e d i k a m e n t ö s ⚒️ 
| Maßnahme                   | Indikation                                                           |
| -------------------------- | -------------------------------------------------------------------- |
| **Langzeit‑O₂**            | PaO₂ <55 mmHg oder <60 mmHg + Cor pulmonale                          |
| **NIV**                    | hyperkapnische Insuffizienz, v. a. bei Exazerbation mit pH ↓         |
| **Lungenvolumenreduktion** | Emphysem                                                             |
| **Sekretmobilisation**     | PEEP, Inhalation, Vibration                                          |

---
## ⏰ VERLAUF
- Langsam progredient, häufige Exazerbationen bestimmen Prognose.  
- Reha verbessert Lebensqualität stärker als jede Einzelmedikation.

---
## 🆘 AKUTE EXAZERBATION

#### 1. Ursachen 
| Ursache              | Details                                       |
| -------------------- | --------------------------------------------- |
| Infekte  (≈80 %) 🦠  | *Rhinoviren*, *H. influenzae*, *Pneumokokken* |
| Umwelt 🌍            | Feinstaub, feuchtkaltes Wetter                |

#### 2. Therapie 
| Therapie   | Maßnahmen               |
| ---------- | ----------------------- |
| O₂         | SpO₂-Ziel: 88–92 %      |
| Inhalativ  | SABA + SAMA             |
| Systemisch | Steroide, ggf. Morphin  |
| Beatmung   | NIV, selten Intubation  |
| Antibiose  | bei Infektkonstellation |
